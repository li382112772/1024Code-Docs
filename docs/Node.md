# 使用说明

1、代码空间基于Linux环境，所以如 `Win32file` 等其他操作系统（如Windows系统）特定的依赖包将无法使用。

2、代码空间运行在一个docker的虚拟环境中，部分命令会受到限制，如Python环境中无法指定用户（命令中带 --user 会报错）。

3、编译和运行命令配置在.1024文件里，若需修改，暂时可以在此文件里按格式修改。（后续会支持可视化配置的能力）。
