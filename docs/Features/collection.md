# 合集

在`轻协同IDE环境`、`AI编程助手`、`编程社区`的基础上，`1024Code` 拥有极佳的交互式编程内容的构建基础。为此，我们对旧版合集功能进行了下升级，用下方几个关键词快速介绍下1024Code的交互式编程合集功能：
- 在同一个界面中制作和预览
- 每一个示例/实例代码都有独立的环境
- 环境切换速度飞快
- 多文件框架级别项目支持
- 得以充分利用的AI助手
