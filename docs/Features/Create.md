# 创建代码空间

<br />

![使用模版创建代码空间](../assets/创建代码空间.png)

### 使用模版创建代码空间

<br />

在个人工作台-主页 或 模板 页面 里选择并点击模板创建代码空间。提交后Fork该模板创建一个新的代码空间，并跳转到IDE界面。

<br />
