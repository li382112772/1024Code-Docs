# 更新日志

## 2023年03月28号

1. 对合集功能进行了优化和升级，可以在IDE环境直接查看合集代码空间，并支持快速切换不同的代码空间；现在您可以尝试打造交互式的编程内容，让其他用户感受交互式编程学习体验。[点击此处查看示例](https://1024code.com/collections/45)
2. ChatGPT 增加了内容流式返回逻辑，卡顿以及返回速度问题得到了解决

## 2023年03月21号

1. 增加了导入本地项目（压缩包）创建代码空间的功能，可以通过导入导出方式和本地项目进行同步了
2. 对`IDE`内`git`代码版本管理、数据库模块的功能细节和提示信息进行了优化，无需阅读文档，按照提示即可直接操作
3. 代码空间详情页中自己发出的评论支持删除了

## 2023年03月01号

1. 集成了真ChatGPT到IDE内（恰逢ChatGPT热度，提前了支持AI能力的计划，大家可以抢先体验一波结合在线IDE和智能AI聊天机器人的编程感受），请在IDE左侧导航栏中选择“ChatGPT聊天窗口”，切换后开始使用
2. 对部分已知问题进行了修复和优化

## 2023年02月17号

1. 编程社区推荐数据增加了随机&相似度排序算法
2. 优化了一版作品详情页（即代码空间详情页），在 `EmbedIDE` 外，可以直接预览 `README` 文件了
3. 调整了复制分享链接功能，给链接拼接了分享文案
4. 修复了编辑器 `editor` 选中内容未高亮的问题
5. 修复了部分稳定性问题

## 2023年02月09号

1. 调整了一版注册功能，保留使用邀请码直接注册和申请参与内测注册方式的同时，开放了加入培育计划注册方式
2. 修复了编辑器缩进模式为空格时，缩进后仍是 `tab` 符的问题
3. 修复非运行按钮运行，通过 `shell` 运行项目时，不监听文件变化的问题

## 2023年01月13号

1. 社区页面增加了空间发布功能，作品分享发布更便捷了
2. `IDE markdown` 文件支持了对项目内图片文件和有序列表的预览
3. 修复了上一版本出现的创建和 `Fork` 代码空间超时的问题

## 2023年01月04号

1. 更新了一版社区功能（完整版本第一版），大家可以[抢先体验](https://1024code.com/community)一波
2. `IDE` 增加了对代码空间容器资源使用率的监控
3. 对部分已知问题进行了修复和优化
